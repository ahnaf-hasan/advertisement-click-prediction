# Advertisement Click Prediction(Logistic Regression)

## Project Overview

This project uses a fake advertising dataset to predict whether an internet user will click on an advertisement displayed on a company website. The goal is to build a Machine Learning classification model that can accurately predict user behavior based on demographic and online activity features.

## Dataset Description

The dataset contains the following features:

| Feature | Description |
|---------|-------------|
| Daily Time Spent on Site | Consumer time spent on the website (minutes) |
| Age | Consumer's age (years) |
| Area Income | Average income of the consumer's geographical area |
| Daily Internet Usage | Average minutes per day the consumer spends on the internet |
| Ad Topic Line | Headline of the advertisement |
| City | Consumer's city |
| Male | Gender of the consumer (1 = Male, 0 = Female) |
| Country | Consumer's country |
| Timestamp | Time when the consumer clicked the ad or closed the window |
| Clicked on Ad | Target variable (0 = Did not click, 1 = Clicked) |
